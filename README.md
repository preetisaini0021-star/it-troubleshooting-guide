# IT Troubleshooting Guide

## About This Project
This project contains simple solutions for common computer and network problems.

## Common Issues & Fixes

### 1. WiFi Not Working
- Restart router
- Check airplane mode
- Reconnect network

### 2. Slow Computer
- Restart system
- Close background apps
- Check for updates

## Tools Used
- Windows OS
- Basic Networking
